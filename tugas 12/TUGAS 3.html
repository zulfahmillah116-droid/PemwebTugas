FAQ Interaktif

Apa itu jQuery? ▶
jQuery adalah pustaka JavaScript yang cepat, kecil, dan kaya fitur. Ini membuat hal-hal seperti penanganan HTML document traversal dan manipulasi, penanganan acara (event handling), animasi, dan Ajax menjadi jauh lebih sederhana dengan API yang mudah digunakan yang berfungsi konsisten di berbagai browser.
Bagaimana cara menggunakan .click() di jQuery? ▶
Metode .click() digunakan untuk melampirkan penangan acara "click" ke elemen HTML yang dipilih. Sintaks dasarnya adalah $("selector").click(function() { ... });.
Apa fungsi dari .toggle()? ▶
Metode .toggle() digunakan untuk menampilkan atau menyembunyikan elemen yang cocok. Jika elemen ditampilkan, elemen tersebut disembunyikan; jika disembunyikan, elemen tersebut ditampilkan. Ini sangat berguna untuk menu dropdown atau, seperti dalam kasus ini, jawaban FAQ.
Apa perbedaan antara .hover() dan CSS :hover? ▶
CSS :hover murni untuk tujuan gaya visual sederhana. .hover() jQuery adalah fungsi pintasan untuk mengikat dua penangan acara: satu untuk mouseenter (saat masuk) dan satu lagi untuk mouseleave (saat keluar), memungkinkan Anda menjalankan logika JavaScript yang lebih kompleks (seperti menambahkan/menghapus kelas dinamis, bukan hanya mengubah warna) saat kursor masuk atau keluar.
Apakah jQuery masih relevan di tahun ini? ▶
Meskipun kerangka kerja JavaScript modern seperti React, Vue, dan Angular lebih populer untuk aplikasi kompleks, jQuery masih digunakan secara luas di jutaan situs web dan sangat relevan untuk proyek yang membutuhkan manipulasi DOM yang cepat dan sederhana tanpa beban kerangka kerja penuh.
